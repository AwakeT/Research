# 单目深度估计 Depth-Anything 技术调研

> **来源文档**：调研文档/Depth-Anything.pdf
> **评审维度**：需求评估与调研评审
> **更新时间**：2026-03

## 一、调研背景与目标

Kinbot导航系统需要可靠的深度感知能力来支撑障碍物检测和空间距离估计。调研Depth-Anything（DA3）单目深度估计模型，评估其在真机后视相机场景下的可用性和精度。

## 二、调研方法

- **论文框架分析**：了解Depth-Anything模型架构与训练方法
- **环境部署验证**：实测cuda12.4环境下的部署流程
- **真机实测**：使用后视相机在多个距离（1m-2.85m）进行深度估计精度测试
- **系统化对比**：前进/后退过程中每0.3m采集深度和RGB进行DA3 vs 真实相机深度对比

## 三、核心调研内容

### 3.1 环境部署

```
pip install xformers torch>=2 torchvision
cd Depth-Anything
pip install -e .  # Basic
pip install --no-build-isolation git+https://github.com/nerfstudio-project/gsplat.git@...
pip install -e ".[app]"  # Gradio, python>=3.10
pip install -e ".[all]"  # ALL
```

### 3.2 真机资源占用

- 模型加载：6.37g
- 推理峰值：8.34g
- 推理用时：0.1s/帧

### 3.3 多距离测试结果

| 距离 | 桌上瓶罐 | 墙体 | 桌腿 |
|------|---------|------|------|
| 1.8m | 2.2x | 2.36 | - |
| 1m | 1.7x | 1.96 | - |
| 2.85m | 2.99-3.10 | 3.16 | 2.76 |

### 3.4 系统化精度评估

- **前进方向**：MAE=0.2071m, RMSE=0.3055m
- **后退方向**：MAE=0.2845m, RMSE=0.3582m
- **近距离场景**（~1m）：平均误差约10cm

## 四、调研结论对规划和方案设计的指导意义

1. **可用性确认**：DA3在1-3m范围内MAE约0.2-0.3m，满足家居场景下的粗粒度距离估计需求
2. **部署可行**：推理速度0.1s/帧、显存峰值8.34g，适合在算力服务器上实时运行
3. **精度边界**：近距离（<1.5m）精度较好，远距离（>2.5m）误差增大，需结合ToF等传感器做多源融合
4. **与ASM集成**：实测结果为后续ASM建图中的深度数据源选择提供了定量依据
